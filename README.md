# ArtisanHub Nairobi

A modern TypeScript web application designed to showcase and connect Nairobi-based artisans with a global audience. This platform highlights traditional Kenyan crafts ranging from soapstone carving to Maasai beadwork.

## Features
- **Artisan Discovery**: Browse through diverse categories of local craftsmanship.
- **Search Functionality**: Filter artisans by name or specific craft type.
- **Global Commerce Focus**: Visualizes price ranges and shipping-ready statuses.
- **Responsive Design**: Mobile-first approach using Tailwind CSS.

## Setup Instructions
1.  **Clone the project**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
4.  **Build for Production**:
    ```bash
    npm run build
    ```

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons
